1. 기본생성자
- 생성자는 객체를 생성하는 역할을 한다.
- 생성자는 객체를 생성할때, 최초로 실행되어야 하는 일을 메소드에 정의할 수 있게 한다.
- 생성자를 사용자가 따로 만들지 않았을 때, 자바의 자체 기능으로 생성자를 만들어 준다.
  → 이것이 기본 생성자 이다.
- 기본생성자 : 클래스와 이름이 같으면서 매개변수가 없는 생성자. 안에 따로 로직이 없는 생성자
- 인스턴스를 만들때의 생성자와 클래스의 생성자는 인자의 유무가 일치해야한다.
- 생성자는 개발자가 복수로 만들 수 있다.
- 생성자를 개발자가 따로 만들었다면, 자바는 기본생성자를 만들어 주지 않는다.

2. super
- 상속을 통해 자식 클래스를 통해 객체를 생성했을 경우, 부모 클래스에 기본 생성자가 없으면 오류가 발생한다.
- 자식클래스와 부모클래스가 동일한 내용의 생성자를 만들었다면, 그것은 중복이다.
- 자식클래스에서 부모클래스의 생성자를 호출하여 동작하게 하면, 위의 문제가 해결될 수 있는데, 그것이 바로 super이다.
- super는 부모클래스의 생성자다(this가 자기 자신을 가리키는 것과 비슷한 맥락이다).
- 자식클래스의 생성자에 추가할 초기화 로직은 super class의 생성자 호출보다 우선해서는 안된다.
