1. overriding
- 부모클래스에게 물려받은 메소드를 재정의해서 새로운 기능을 발휘할 수 있게 하는 것이 overriding이다.
- 자식클래스에 부모클래스의 메소드를 새로 입력하여 사용하고 싶은 기능을 구현할 수 있다.
- overriding된 메소드는 자식클래스가 우선순위를 가지고 자식클래스의 메소드가 실행되게 된다.

2. overriding의 조건
- overriding한 자식클래스의 메소드는 부모클래스의 메소드와
1)이름이 같아야 한다.
2)데이터 타입이 같아야 한다.
3)매개변수의 개수, 데이터 타입, 순서가 같아야 한다(있는 경우).
