1. 데이터베이스의 본질
- 데이터베이스의 입력과 출력을 알면 데이터베이스의 반은 안다고 할 수 있다.
- 입력 : 데이터 생성, 수정, 삭제 / 출력 : 데이터 읽기
- 즉 데이터 베이스의 핵심은 CRUD(Create, Read, Update, Delete)이다.
- 파일 → 스프레드시트 → 데이터베이스 순으로 데이터베이스의 관리를 쉽고 더욱 복잡한 작업까지 가능하게 한다.
- 데이터베이스의 최신을 알기 위해서는 datebase ranking 2022처럼 검색을 해보라
- 데이터베이스는 여러 종류가 있지만, 현재 관계형 데이터베이스 관리 시스템(Relationship DBMS)이 주도하고 있다.
- 데이터베이스를 공부할 때, 관계형 데이터베이스를 공부하고, 관계형 데이터베이스가 아닌 데이터베이스를 공부하는 것을 추천한다.

2. 데이터베이스의 목적
- 스프레드 시트는 데이터를 관리할 때 클릭을 통해 여러가지 클릭을 해야한다. 
- 데이터베이스 프로그램은 약정된 문법을 통해 시스템에 명령을 하면 요청한 결과를 출력한다.
