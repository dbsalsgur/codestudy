1. 공유기(Router)
- 두대의 컴퓨터가 통신하기 위해서는 IP주소가 필요하다
- 스마트폰, 노트북 등 한 가정에 여러대의 컴퓨터가 있을 경우 각각의 기계가 인터넷 신호를 필요로 하는데, 여러개의 인터넷을 가입하면 비싸다. 이 때 필요한 것이 공유기이다.
- 원래 사용하던 인터넷 선을 공유기에 연결하면 부여된 ip주소로 요청을 했을 때, 컴퓨터가 아닌 공유기에 접속을 하게된다.
- 공유기에는 Wan이라는 포트와 Lan이라는 포트가 있다.
- LAN : Local Area Network, 지역 네트워크
- WAN : Wide Area Network, 광범위 네트워크, 공유기가 인터넷이라고 하는 전 세계가 접속할 수 있게 하는 네트워크에 연결될 수 있도록 하는 포트
- 여러개의 컴퓨터가 공유기를 통한 주소를 부여받을 때, ip주소를 새로 부여받는다(기존 인터넷에 의한  ip가 아님)  ex) 스마트폰은 192.168.0.2, 데스크탑은 192.168.0.3 이런 식
- 공유기에도 ip주소가 부여되는데 이를 Gateway address, Router Adress라고 한다
- public ip adress : 전세계와 연결되어 있는 인터넷 주소(가입한 인터넷 선꼽으면 부여되는 거)
- private ip adress : 회사나 가정내에서만 쓰이는 주소(공유기 연결하면 부여되는 거)
- private ip주소로 약속된 범위
10.0.0.0 ~ 10.255.255.255
172.16.0.0 ~ 172.31.255.255
192.168.0.0 ~ 192.168.255.255

2. NAT(Network Address Translation)
- 192.168.0.4 컴퓨터에서 웹페이지 접속 시(공유기 有) 공유기로 1차로 신호를 보냄.
- 공유기에서는 192.168.0.4 라는 ip주소를 가진 컴퓨터가 보낸 신호라고 내부적으로 기록
- 192.168.0.4는 외부에서 접속불가(사설 ip주소이기 때문에), 이 때 공유기의 NAT라는 기능이 작동함
- NAT는 192.168.0.4라는 주소를 공유기를 설치하기 전 주소인 공용 ip주소로 변환해서 해당 웹페이지에 접속함
- 똑같이 해당 웹페이지에서 정보를 수신할 때 기존 공용 ip로 데이터를 수신하여 공유기는 기기의 원래 사설 ip로 변환하여 전달한다.
- 위 내용은 클라이언트 컴퓨터가 동작할 때의 과정이다.

3. ip주소 알아내기
- 웹페이지에 라우터의 ip주소를 입력하면 관리자모드에 진입 가능
- 외부 ip가 공용 ip, 내부 ip가 사설 ip이다.

4. port
- 외부의 다른 사람이 서버 컴퓨터에 접속해서 응답해줄 수 있는가?
- 한 컴퓨터 내에는 여러개의 서버가 있을 수 있다. 
- 이 때 클라이언트 컴퓨터가 어떤 서버와 상호작용할 지 컴퓨터에 알려줘야 한다. 그래서 port에는 번호가 있고, port에 서버를 개설할 수 있다.
- port는 총 65535개가 있다.
- 그 중 0번부터 1023번까지는 Well-known port이며 마음대로 쓸 수 없는 포트이다.
- 만약 내 컴퓨터에 서버를 설치하면 80번 port를 사용(Listening)하기로 약속 되어있다.
- 컴퓨터에 서버를 하나 더 설치하면 다른 포트에 해야하는데 관습적으로 8000번이나 8080번 port를 사용한다.
- 접속하고자 하는 서버를 선택하는 것은 URL에 따른다.
ex) http://opentutorials.org는 80번 port에 접속되도록 약속되어 있다.
8080port에 접속하려면 http://opentutorials.org.8080으로 접속하면 된다.

5. port forwarding
- 192.168.0.4라는 ip에 서버를 설치했다고 가정하자
- 해당 서버에 접속하기 위해서는 공용 ip주소를 통해 접속해야하는데, 이 때 192.168.0.4라는 사설 ip로 연결할 수 있는 방법, 그것이 port forwarding이다.
ex) 내 컴퓨터의 공용 ip가 59.6.66.238, 서버가 각각 192.168.0.3과 192.168.0.4에 설치되있을때
59.6.66.238:8080 → 192.168.0.3:80
59.6.66.238.8081 → 192.168.0.4:80
으로 공유기 관리자에서 외부ip와 내부ip를 각각 설정하면 저렇게 접속된다.
